My Generator QP v0.1.0 Release Notes
==========================

* New features
- Интерактивный UI на PyQt5: добавлены поля параметров (`N`, `T`, `dt`, `шум %`) и две кнопки — «Генерировать QP и график» и «Сохранить QP».
- Визуализация: построение ступенчатого задания и реального положения с помощью matplotlib.

* Tweaks
- Интеграция UI с симулятором: корректные слоты без аргументов; после генерации активируется кнопка сохранения.
- Симуляция сигнала: инерционное звено с настраиваемыми `T`, `dt`, уровнем шума и числом точек; автоматическая генерация временной оси и ступенчатого задания.
- Экспорт данных: сохранение результатов симуляции в CSV через `Simulator.save_to_csv()`.
- Файловая структура вывода: результаты сохраняются в `src/generator_qp/src/QP_out/out_merge.csv` (директория создаётся автоматически).
